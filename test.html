Chat Widget Test Page

This is a test page to demonstrate the chat widget functionality. The chat widget should appear as a floating button in the bottom-right corner of the page.

Configuration Options

The chat widget can be configured by setting window.ChatWidgetConfig before loading the widget script:

<script>
window.ChatWidgetConfig = {
    title: "Support Chat",                    // Header title
    logo: null,                              // Optional logo URL
    initialMessage: "Hi! How can I help?",   // First message shown
    color: "#28a745",                        // Primary color (header, buttons)
    userMessageBgColor: "#28a745",           // User message background
    userMessageTextColor: "#ffffff",         // User message text color
    assistantMessageBgColor: "#f1f3f4",     // Assistant message background
    assistantMessageTextColor: "#333333",   // Assistant message text color
    apiURL: "http://localhost:3000/messages" // API endpoint
};
</script>

API Requirements

The chat widget expects a POST endpoint that:

  • Accepts JSON payload: {"threadId": "string", "content": "string"}
  • Returns JSON response: {"content": "string"}
  • For testing, start a server at http://localhost:3000/messages

How to Use

  1. Include the configuration script (as shown above)
  2. Include the CSS file: <link rel="stylesheet" href="chat-widget.css">
  3. Include the JavaScript file: <script src="chat-widget.umd.cjs"></script>
  4. The widget will automatically initialize when the page loads

Click the chat button in the bottom-right corner to test the widget!

Note: For the chat to work properly, you need to have a backend server running at the configured API URL that can handle the message requests.
